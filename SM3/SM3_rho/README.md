1.项目简介：使用python实现对于SM3算法的rho攻击

2.项目名称：

SM3_rho.py

3.具体代码说明：

为了追求更高的效率，本项目使用了gmssl库中的相关函数实现对任意字符串加密，并对其进行rho攻击并测量攻击所需要的时间。

4.运行指导：直接运行本项目并且根据提示输入攻击长度即可获得两个随机生成的消息以及碰撞和所需时间
